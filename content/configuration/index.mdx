# ⚙️ Configuration

## Overview
Oraxen allows you to create a wide variety of custom content for your Minecraft server. This section covers everything you need to know about creating them.

## Tips
### Organization
Maintaining Organization is heavily recommended for future changes (especially if your Workspace gets bigger over time).
