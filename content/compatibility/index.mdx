# 🤝 Compatibility

## Overview
Oraxen integrates into many different plugins & software, it also doesn't cause any issues with most popular or more simpler plugins.
If you want to let users know that your or another plugin / software is compatible, consider adding your page through <Highlight color="#1877F2" to="https://github.com/oraxen/docs.oraxen.com">**GitHub**</Highlight>.
