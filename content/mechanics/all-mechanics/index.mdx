# All Mechanics

Oraxen provides a wide variety of mechanics that give special abilities to your items. These are organized into categories:

## [Miscellaneous](./misc)

General-purpose mechanics for various item behaviors:
- **Custom Food** - Custom hunger/saturation values and effects
- **Backpack** - Turn items into portable storage
- **Music Disc** - Custom music discs with custom sounds
- **Durability** - Custom durability values
- **Efficiency** - Mining speed boosts
- **Consumable** - Make items consumable on right-click
- **Repair** - Items that repair other items
- **Commands** - Execute commands on item use
- **Armor Effects** - Potion effects when armor is equipped
- **Aura** - Particle effects when holding items
- **Hat** - Make any item wearable as a hat
- **Soulbound** - Keep items on death
- **Skinnable/Skin** - Change item textures dynamically
- **Dimmable Light** - Cycle through light levels when right clicking.

## [Combat](./combat)

Weapon and spell mechanics for combat:
- **Thor** - Throw lightning bolts
- **Lifeleech** - Steal health from enemies
- **Bleeding** - Apply bleed damage over time
- **EnergyBlast** - Particle cone attack
- **Witherskull** - Launch wither skulls
- **Fireball** - Launch exploding fireballs

## [Farming](./farming)

Tools for farming and mining:
- **Harvesting** - Auto-harvest and replant crops
- **BigMining** - Mine multiple blocks at once (3x3, 5x5, etc.)
- **Smelting** - Auto-smelt ores when mined
- **BottledExp** - Convert experience to bottles
- **BedrockBreak** - Break bedrock blocks
- **Watering** - Watering can system for farms

## Special Mechanics

Some mechanics have their own dedicated pages:
- [Custom Mechanic](./custom-mechanic) - Create custom event-based mechanics
- [ClickAction](./clickaction-mechanic) - Run actions when clicking blocks/furniture
- [Furniture Mechanic](/mechanics/furniture-mechanic) - Custom furniture and decorations
- [NoteBlock Mechanic](/mechanics/noteblock-mechanic) - Custom blocks using noteblocks
- [StringBlock Mechanic](/mechanics/stringblock-mechanic) - Custom blocks using tripwire
