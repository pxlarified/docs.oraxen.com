# 🔧 Developers

## Overview
Oraxen is built around expansion and has an extensible API which can be used to add compatibility for a service / software.
It is entirely open source and allows you to make changes to the actual plugin, not just yours, if you need something changed.

## Support
Oraxen will provide Developers with access to support to answer questions that are related to your integration (e.g. an issue with the API).
You can obtain support on our <Highlight color="#1877F2" to="https://discord.gg/sstfjsJcQ5">**Discord**</Highlight>.

<Callout type="info">
  Maintainers and developers of Oraxen will also be happy to help you with the implementation of your integration (such as Tips, Debugging, Assisting with e.g. Coding).
</Callout>
