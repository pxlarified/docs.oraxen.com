# 📘 Introduction

Hey there and welcome to the Oraxen documentation.

## Overview
Oraxen allows you to customize your Minecraft server by allowing the creation of new blocks, items, and recipes. 
This can be accomplished entirely through a resource pack and server-side implementation, eliminating the requirement for any client-side mod installations

## How It Works
Oraxen reads your configuration files and uses them to generate a minecraft resource pack that will then be uploaded to a resource pack server which then sends the pack to players joining your server.
Polymath is a free and opensource software written in Python by Oraxen to host Minecraft resourcepacks. Oraxen supplies you with a free Polymath instance, hosted in Switzerland on an Oracle Virtual Private Server.

## Version
Oraxen is developed and designed for the <Highlight color="#1877F2" to="https://papermc.io/downloads/paper">**Paper**</Highlight> Server Software. It is compatible with most common forks including <Highlight color="#1877F2" to="https://papermc.io/software/folia/">**Folia**</Highlight>. Oraxen is also compatible with the <Highlight color="#1877F2" to="https://mcserverjars.com/spigot">**Spigot**</Highlight> Server Software, however our changes are not actively tested against it. When using 1.21.3 or lower, please install Oraxen version 1.183.0 to get compatible default assets for your version. After that you can upgrade Oraxen to the latest version.

## Installation
Installing Oraxen is a fairly straight forward process. Simply drag and drop the .jar file into your server's /plugins folder and restart your server.

<Callout type="info">
  Oraxen requires CommandAPI and PacketEvents (or Protocollib) to function. Since Version 1.203.1, Oraxen automatically downloads CommandAPI and PacketEvents when not already installed.
</Callout>
