# ❓ FAQ

### Can I Disable The Default Assets And Configs Oraxen Comes With?

Absolutely. Your `settings.yml` file contains options to disable both of these.
Note: Required configs and assets will still be generated, but most will be disabled.

### Can I Use Oraxen Items In Custom Recipes?
Yes, Oraxen items can be used in both vanilla crafting recipes and custom recipes. You can reference Oraxen items in recipe configurations using their item IDs. Check the recipes documentation for detailed examples.

### Can I Use Oraxen With Other Plugins?
Yes, Oraxen is designed to be compatible with most plugins & allows easy integration for plugin developers. However, some plugins that heavily modify items or resource packs might conflict. For a detailed list of compatibility check out our <Highlight color="#2E8B57" to="./compatibility">**Compatibility**</Highlight> section.

### Does Oraxen Replace Any Vanilla Items?
The goal of Oraxen is to add things to the game without losing features, so the short answer is no. However, Minecraft has some limitations.

### Does Oraxen Work With Modded Clients?
Oraxen works with vanilla Minecraft clients and most modded clients that support resource packs. However, some mods might interfere with resource pack loading or custom model data (e.g. Optifine or Iris Shaders). Fabric/Forge clients generally work well with Oraxen. We will signalize issues with well-known mods that interfere with resource packs in our docs (Optifine & Iris both work fine).

### How Do I Backup My Oraxen Configuration?
Simply backup your entire `/plugins/Oraxen/` folder, which contains all configurations, items, mechanics, and generated resource pack files. It's recommended to backup before major updates or configuration changes.

### How Do I Disable Or Change The Welcome Sound?
You can configure this in `settings.yml`. The example below shows where you can disable or change the sound.

```yaml
  receive:
    enabled: true
    loaded:
      actions:
        sound:
          enabled: true
          type: minecraft:welcome
          volume: 1.0
          pitch: 1.0
```

### How Do I Hide Item Tooltips?
Use the <Highlight color="#1877F2" to="https://github.com/lolgeny/item-tooltip-remover">**Item Tooltip Remover**</Highlight> resource pack or datapack to hide tooltips for custom items.

### How Do I Just Use Oraxen's Mechanics (no resource pack)?
Go to your `settings.yml` file and set the following options.

```yaml
  upload:
    enabled: false
Pack:
  generation:
    generate: false
    compression: BEST_COMPRESSION
    protection: false
  dispatch:
    send_pack: false
  enable_configs_updater: false
Misc:
  reset_recipes: false
  auto_update_items: false
```

The configurations do not go in that exact order. Once you've changed these lines, delete what is inside the `/Oraxen/pack`, `/Oraxen/items` and `/Oraxen/glyphs` paths and restart the server.

### How Do I Optimize Oraxen Performance?

- Use texture atlasing to reduce the number of texture files
- Optimize texture file sizes (use appropriate resolutions)
- Disable unused mechanics in `settings.yml` and `mechanics.yml`
- Use compression for resource pack generation
- Limit the number of custom model data variations per item

### How Do I Troubleshoot Resource Pack Issues?

1. Check the server console for Oraxen errors or warnings
2. Check the client logs (the logs of your minecraft client) for the reason why the pack is having issues
3. Remove the `/plugins/Oraxen/pack/pack.zip` file and reload.

### How Do I Update Oraxen?
1. Backup your current Oraxen folder
2. Download the latest version
3. Replace the old jar file with the new one
4. Restart your server
5. Check for any configuration updates or breaking changes in the changelog

A helpful video guide can be found <Highlight color="#e74c3c" to="https://youtu.be/LkansZwVaPY">**here**</Highlight>.

### I Use Bungee/Velocity But The Pack Keeps Reloading When Players Switch Servers?
This happens because players technically leave one server and join another, causing Minecraft to remove and resend the resource pack.
To prevent this, use <Highlight color="#1877F2" to="https://www.spigotmc.org/resources/%E2%9C%82%EF%B8%8F-bungee-pack-layer-optimize-resource-pack-sending.94978/">**BungeePackLayer**</Highlight>, a Bungee/Velocity plugin that prevents unnecessary pack resending. Note that packs will still be resent if they differ between servers (when config files aren't identical). Other optional plugins are <Highlight color="#FFA500" to="https://modrinth.com/plugin/bungeeresourcepacks">**BungeeResourcepacks**</Highlight>, <Highlight color="#2E8B57" to="https://modrinth.com/plugin/velocityresourcepacks">**VelocityResourcepacks**</Highlight> or <Highlight color="#1877F2" to="https://modrinth.com/plugin/worldresourcepacks">**WorldResourcepacks**</Highlight>

### Is Oraxen a Mod?
No, Oraxen is not a mod. It's a Minecraft plugin that adds custom items, blocks, and features to the game using only a resource pack and server-side implementation. Players don't need to install any client-side modifications.

### Oraxen Is Using Its Own Resource Pack, Can I Still Use Mine?
Yes, you can merge your existing resource pack with Oraxen's generated pack. More information about pack merging can be found <Highlight color="#1877F2" to=".usage/merging">**here**</Highlight>.

### What File Formats Does Oraxen Support For Textures?
Oraxen supports PNG files for textures and JSON files for models. Textures should be in PNG format with appropriate resolutions (16x16, 32x32, etc.). Models should follow Minecraft's JSON model format.

### Adding an item breaks the Textures Of Others Already Created?
By default (pre 1.21.4), Oraxen automatically picks custom model data values for your items. Every item with a different model needs a unique custom model data value. When manually setting custom model data values, you might accidentally use the same value for different items, causing conflicts.

Don't forget to reload the plugin with `/o reload all`.

### Why Do My Textures Work When I Use OptiFine But Not In Vanilla?

Since Minecraft 1.11, vanilla resource packs don't support uppercase letters in folder, texture, or model names. OptiFine still supports uppercase, but you should always use lowercase letters and underscores to avoid compatibility issues.
